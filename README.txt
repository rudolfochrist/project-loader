1 NAME
======

  project-loader --- A utility to load Quicklisp systems without
  Quicklisp itself


2 VERSION
=========

  ,----
  | 0.7
  `----


3 SYNOPSIS
==========

  ,----
  | CL-USER> (project-loader:pin-dependencies)
  `----

  Later..

  ,----
  | $ sbcl --no-userinit --load init.lisp
  `----


4 DESCRIPTION
=============

  Dump images or create executables without Quicklisp but still find
  your dependencies.
